The Node.js framework for all web developers with everything what you need.

Instructions:

  • Add backend URL to config
  • All backend routes are inside controllers/api.js
  • All API are in schemas/example.js
  • All AJAX requests to backend are inside public/js/example.js
  • If you would like to add more attributes into the request, you'll need to specify them in schema schema.define('newparameter', type) in schemas/example.js.
  • You can check model coming into schema with console.log($.model)
  • New attribute needs to be added also in public/js/example.js
